// Introduction

Welcome to LukiLinux

LukiLinux is a lightweight Linux distribution built on top of Ubuntu. The idea is simple: take a solid, well-supported base, strip out everything that doesn't need to be there, and end up with something that's fast, reliable, and out of your way.

If you've used Ubuntu before, you'll feel right at home. Everything you know — apt, the terminal, .deb packages — all works exactly the same. We just removed the parts nobody asked for.

Note: LukiLinux is a personal project and is actively being developed. Things may change between releases.

// Requirements

System Requirements

LukiLinux is designed to run well on older and lower-end hardware. Here's what you need:

  • CPU: Any 64-bit (x86_64) processor
  • RAM: 1 GB minimum — 2 GB or more recommended
  • Storage: At least 10 GB free space (HDD is fine, SSD will be faster)
  • Desktop: XFCE 4.18
  • USB Drive: 4 GB or more (for installation media)
  • Internet: Recommended but not required to install

If your machine is from the last 15 years and runs 64-bit, it should handle LukiLinux fine.

// Installation

Installing LukiLinux

Step 1 — Download the ISO

Head over to the Download page and grab the latest stable ISO.

Step 2 — Flash it to a USB

Use Balena Etcher (recommended for beginners) or dd from the terminal.

# Linux/Mac terminal method:
sudo dd if=lukilinux-1.0.iso of=/dev/sdX bs=4M status=progress
Warning: Replace /dev/sdX with your actual USB device. Double-check with lsblk first — dd will overwrite whatever is there.

Step 3 — Boot from USB

Restart your machine and boot from the USB. You'll usually do this by pressing F12, F2, or Delete at startup to open the boot menu — it varies by manufacturer.

Step 4 — Run the Installer

The graphical installer will walk you through partitioning, setting up your user account, and choosing your timezone. It's straightforward — just follow the prompts.

Step 5 — Reboot and enjoy

Remove the USB when prompted, reboot, and you're in. First boot takes a moment while things settle — after that it'll be fast every time.

// Post-Install

First Steps After Installing

Once you're in, here are a few things worth doing right away.

Update everything

sudo apt update && sudo apt upgrade -y

Install a browser

# Firefox:
sudo apt install firefox

# Or Chromium:
sudo apt install chromium-browser

Set up your locale and timezone

sudo dpkg-reconfigure locales
sudo dpkg-reconfigure tzdata

// Software

Installing Software

LukiLinux uses apt, the same package manager as Ubuntu. If you've ever used Ubuntu or Debian, it's exactly what you know.

Search for a package

apt search package-name

Install a package

sudo apt install package-name

Remove a package

sudo apt remove package-name

Install a .deb file manually

sudo dpkg -i filename.deb

You can also add PPAs (Personal Package Archives) for software that isn't in the default repos, same as you would on Ubuntu.

sudo add-apt-repository ppa:some/ppa
sudo apt update
sudo apt install the-package

// Updates

Keeping LukiLinux Updated

Since we're built on Ubuntu LTS, you'll get security updates regularly. It's a good habit to update every week or so.

sudo apt update && sudo apt upgrade -y

To also clean up old packages that are no longer needed:

sudo apt autoremove && sudo apt autoclean
Tip: Full distro upgrades (e.g. jumping to a new LTS version) will be announced on this site when they're available.

// FAQ

Frequently Asked Questions

Is this based on Ubuntu?

Yes. LukiLinux is built on Ubuntu LTS. That means full compatibility with Ubuntu packages, PPAs, and documentation.

What desktop environment does it use?

That'll be specified on the download page per release. The minimal edition ships with none — you install what you want.

Can I install it alongside Windows?

Yes, dual booting works fine. The installer will give you the option to install alongside an existing OS. Just make sure you have backups before messing with partitions.

Is it free?

Completely. No cost, no catch, no premium tier. Just Linux.

Where do I report bugs or ask questions?

The project is a work in progress — a contact page or GitHub issues link will be added here soon.
